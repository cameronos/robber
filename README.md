### Installation
- Drag the "robber" folder" into the resource folder of your FXServer.
- Edit your server.cfg and add "start robber" or "ensure robber" to the resource list.

### Usage
/robber location<br>
e.g. /robber Marina Dr.<br>
"Robber at Marina Dr"
