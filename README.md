# robber
FiveM Lua Script that prints out a DrawNotification

# Installation
1. Drag the robber folder into the resource folder of your FXServer folder.
2. Edit your server.cfg and add "start robber" to the resource list.

# Usage
/robber location
Example:
/robber Marina Dr
Robber at Marina Dr
